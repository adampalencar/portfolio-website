    - For each spacing token (--space-1 through --space-9), render a horizontal bar:

      - Bar height: 24px
      - Bar color: var(--accent-15)
      - Label to the right: variable name + pixel value in `.mono-label` (e.g., "--space-5 / 32px")

    **Section 4: Button States**
    - Section label: "04 / BUTTONS" using `.mono-label`
    - Two rows: "Filled" and "Ghost"
    - Filled buttons: background var(--accent), color white, padding var(--space-2) var(--space-4), border-radius 6px, font-family var(--font-body), font-size var(--size-sm), font-weight 500
      - States: Rest, Hover (background var(--accent-hover)), Focus (outline 2px solid var(--accent), outline-offset 2px), Disabled (opacity 0.5, cursor not-allowed)
      - Show all 4 states side by side. Hover and Focus states are forced via CSS classes (`.dev-btn--hover`, `.dev-btn--focus`) so they're visible without interaction
    - Ghost buttons: background transparent, border 1px solid var(--accent), color var(--accent), same sizing
      - Same 4 states, hover state gets background var(--accent-5)

    **Section 5: Grain Texture**
    - Section label: "05 / GRAIN TEXTURE" using `.mono-label`
    - Two large rectangles side by side (each ~300px wide, ~200px tall):
      - Left: background var(--bg-primary), label "bg-primary + grain"
      - Right: background var(--bg-secondary), label "bg-secondary + grain"
    - The grain overlay from BaseLayout covers these naturally. Include a note: "Grain overlay is global at 4% opacity via position:fixed in BaseLayout"

    **Section 6: Dark Mode Toggle**
    - Section label: "06 / THEME TOGGLE" using `.mono-label`
    - A button that toggles data-theme between "light" and "dark" on document.documentElement
    - The button also saves the choice to localStorage under key "theme"
    - Button text shows current state: "Current: Light - Click to switch to Dark" (and vice versa)
    - Use an inline script (not is:inline, a regular Astro `<script>` block) for the toggle logic:
      ```js
      const btn = document.querySelector('[data-theme-toggle]');
      function updateLabel() {
        const current = document.documentElement.dataset.theme || 'light';
        btn.textContent = `Current: ${current.charAt(0).toUpperCase() + current.slice(1)} — Click to switch to ${current === 'light' ? 'Dark' : 'Light'}`;
      }
      btn?.addEventListener('click', () => {
        const next = document.documentElement.dataset.theme === 'dark' ? 'light' : 'dark';
        document.documentElement.dataset.theme = next;
        localStorage.setItem('theme', next);
        updateLabel();
      });
      updateLabel();
      ```
